abstract: >-
  <span class="lang-en">Finding a lattice geometry that produces a wanted vibrational response is a one-to-many problem, and each candidate still has to go through a finite-element solve before you can check it. A workable design pipeline needs two networks. One maps a geometry to its response quickly; the other proposes geometries for a requested response. Much of the published deep-learning work keeps the problem small by reducing the response to a band-gap width and a center frequency. This poster covers the forward half, trained on the full band structure rather than those two numbers.<br><br>The model reads the unit cell as a graph: lattice junctions become nodes, beams become edges. Node and edge features are lifted to 768 dimensions and run through six GINE message-passing layers, which let each junction draw on information up to six nodes away. Ten learned band queries then pool the nodes by attention, one query per band, producing a vector per band. A short residual stack of 1D convolutions along the wavevector axis mixes the bands and smooths the dispersion. The result is ten bands at 311 k-points along Y&Gamma;XM&Gamma;, each carrying a frequency and a three-way polarization split (one longitudinal, two transverse). One pass replaces the eigenvalue solve.<br><br>Preparing healthy data is difficult, label bands are not the same from one structure to the next. The fix is to match predicted and true bands optimally before measuring any error, with a Hungarian assignment recomputed at every step. The model is then graded on which curves it produced, not on the order it put them in. Extra terms on correlation, slope and amplitude along k keep the top bands from collapsing, and a second head reads polarization off the same match.<br><br>This forward model is the groundwork for the next piece: a graph VAE that runs the map backwards, from a target band structure to a geometry, with the same approach later extended to other physical properties.</span><span class="lang-fr">Trouver une g&eacute;om&eacute;trie de treillis qui donne une r&eacute;ponse vibratoire vis&eacute;e est un probl&egrave;me one-to-many, et chaque candidat doit encore passer par un calcul par &eacute;l&eacute;ments finis avant d'&ecirc;tre valid&eacute;.
